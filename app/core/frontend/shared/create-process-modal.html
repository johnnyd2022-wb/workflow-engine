What are process steps?

Process steps allow you to define step-by-step workflows that automate your inventory management and provide complete source-to-sale traceability. Each process step represents a physical operation in your manufacturing workflow.

Examples: In food manufacturing, you might have steps like "Mixing", "Fermentation", "Packaging". In pharmaceuticals, steps could be "Active Ingredient Mixing", "Tablet Pressing", "Quality Control". In cosmetics, steps might include "Emulsion Creation", "Filling", "Labeling".

Use the name you already call this process internally. This helps your team recognize the step immediately.

Provide a high-level summary of what this process step involves. This helps others understand the step's purpose.

Define Inputs

Inputs are the physical materials you add to this step. Choose a tab below: add from items you already have in stock, or define inputs not in inventory.

Select from items you have in stock. Define quantity and unit; at execution you choose which batch to consume.

Define other materials used during this step in your process that are not items you keep in your inventory (e.g. water, freshly foraged food). Set default quantity and unit or prompt at execution.

Define Outputs

Outputs are the products or materials produced by this step. These can be used as inputs in subsequent steps, creating a complete workflow chain.

Example Workflow:

Step 1 "Mixing": Outputs "Mixed Base" → Step 2 "Fermentation": Uses "Mixed Base" as input, outputs "Fermented Product" → Step 3 "Packaging": Uses "Fermented Product" as input, outputs "Final Product"

Industry Examples: In pharmaceuticals, a "Tablet Pressing" step might output "Compressed Tablets". In food manufacturing, "Baking" outputs "Baked Goods". In cosmetics, "Emulsion" outputs "Cream Base" that can be used in "Filling" steps.

Execution Prompts

Execution prompts are custom fields recorded when this step runs. Use the Batch number field below for traceability (recommended), and add other prompts for quality control, measurements, or execution-specific information.

Recommendation: Use the Batch number field below for traceability. For other prompts, you can add e.g. Temperature (number with unit "°C"), Quality grade (select), Operator name (text), Pressure or viscosity (number with unit). These values are stored with each execution and used for traceability and reporting.

Finish Creating Process

By finishing this process, you are locking in the process definition.

Once you execute this process, your inventory will automatically update in real-time. Each execution will consume inputs and produce outputs according to the steps you've defined.

You can edit this process at any time by expanding the process steps in the process view. All changes will apply to future executions.

Press "Finish creating process" to save and lock in this process, or select "Back" if you want to edit a step or add another step.
