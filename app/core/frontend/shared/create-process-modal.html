What are Custom Processes?

Custom processes allow you to define step-by-step workflows that automate your inventory management and provide complete source-to-sale traceability. Each process step represents a physical operation in your manufacturing workflow.

Examples: In food manufacturing, you might have steps like "Mixing", "Fermentation", "Packaging". In pharmaceuticals, steps could be "Active Ingredient Mixing", "Tablet Pressing", "Quality Control". In cosmetics, steps might include "Emulsion Creation", "Filling", "Labeling".

Use the name you already call this process internally. This helps your team recognize the step immediately.

Provide a high-level summary of what this process step involves. This helps others understand the step's purpose.

Define Inputs

Inputs are the physical materials you add to this step. These can be raw materials, ingredients, or outputs from previous steps.

+ Add from inventory

Select from existing inventory items. Use this when you have materials already in stock (e.g., raw materials from suppliers, outputs from previous steps). You'll define the quantity and unit here, and at execution time you'll choose which specific batch to consume from your inventory.

+ New Input

Define a new input that will be manually entered at execution time. Use this for materials not yet in inventory or for variable quantities (e.g., water, specific ingredient amounts that change per batch). You can set a default quantity and unit, or choose to be prompted at execution.

Define Outputs

Outputs are the products or materials produced by this step. These can be used as inputs in subsequent steps, creating a complete workflow chain.

Example Workflow:

Step 1 "Mixing": Outputs "Mixed Base" → Step 2 "Fermentation": Uses "Mixed Base" as input, outputs "Fermented Product" → Step 3 "Packaging": Uses "Fermented Product" as input, outputs "Final Product"

Industry Examples: In pharmaceuticals, a "Tablet Pressing" step might output "Compressed Tablets". In food manufacturing, "Baking" outputs "Baked Goods". In cosmetics, "Emulsion" outputs "Cream Base" that can be used in "Filling" steps.
