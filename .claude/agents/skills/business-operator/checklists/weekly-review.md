# Weekly Review Checklist (Business Operator)

Run this at the start of each week (or Monday cron). Tick as you go.

## Gather
- [ ] Read `context/operating-principles.md` + `context/founder.md` (Current goals).
- [ ] Load active projects: `projects/whistlebird/ACTIVE_PROJECTS.md`, `projects/bize/ACTIVE_PROJECTS.md`.
- [ ] Load pipelines: `projects/*/SALES_PIPELINE.md`.
- [ ] Load content calendars: `projects/*/CONTENT_CALENDAR.md`.
- [ ] List deadlines in the next 2–3 weeks (compliance, awards, distributor).
- [ ] Confirm **this week's real capacity** (nights available, fixed commitments, energy). Ask if unknown.

## Assess
- [ ] Any sales follow-up silent >1 week? → Top-3 candidate.
- [ ] Any marketing/content silent >1 week? → Top-3 candidate.
- [ ] Any deadline with an external dependency this week/next?
- [ ] Any "project" that has become repeatable operations? → recommend Biz-E.
- [ ] Score candidates by leverage ÷ effort.

## Decide & schedule
- [ ] Pick Top 3; assign business + specialist skill to each.
- [ ] Time-block Top 3 against actual capacity (don't overfill).
- [ ] Explicitly defer the rest with a revisit date.
- [ ] Choose tonight's single next action (≤30 min).

## Emit
- [ ] Fill `templates/weekly-command-centre.md` → save to `outputs/weekly-command-centre.md`.
- [ ] Issue handoffs to Sales Manager / Marketing Director / Content Producer / Project Manager.
