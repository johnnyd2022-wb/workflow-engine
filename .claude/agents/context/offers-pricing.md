## Whistlebird — unit economics

| Item | Value |
|---|---|
| Bottle size | 700 ml |
| ABV | ~44% |
| Case size | 6 bottles |
| Wholesale price (latest) | **~$47.47 + GST** / bottle |
| Wholesale price (earlier refs) | $46.25, $49.56 + GST |
| Cost to produce | **~$25.55 / bottle after GST** |
| GST | 15% |
| Profit tax assumption | 28% |

**Indicative per-bottle gross margin** (illustrative — confirm the correct GST/tax
treatment before relying on it): wholesale ~$47.47 − cost ~$25.55 ≈ **~$21.9 gross/bottle**,
≈ **~$131 gross per 6-bottle case**, before tax and overhead. Treat as a planning figure,
not an accounting figure.
